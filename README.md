Proyecto 1 del libro " Python Crash Course: A Hands-on, Project-based Introduction to Programming" 
Este proyecto es la creación de un videojuego llamado "Alien Invasions" que es una imitación del videojuego "Space Invaders".

Requisitos:
- Es necesario tener instalado Python3
    python --version
- El paquete pygame
    python-m pip install pygame

Para ejecutar el juego:
- Abrir la terminal dentro de la carpeta alien-invasion
- Usar el comando
    python .\alien_invasion.py
